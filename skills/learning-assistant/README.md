# 🎓 Claude Code 学习助手技能包

一个专为编程初学者设计的 Claude Code Skill，帮助你用通俗易懂的方式学习编程、理解代码、掌握概念。

## ✨ 特点

- 📖 **通俗易懂**：用日常生活的比喻解释复杂的编程概念
- 🎯 **逐步引导**：从简单到复杂，循序渐进的教学方式
- 💡 **实用示例**：提供可直接运行的代码示例
- 🤝 **友好互动**：耐心、鼓励性的学习氛围
- 🔍 **深入浅出**：既讲"是什么"，也讲"为什么"

## 🚀 快速开始

### 安装方法

复制 `.claude` 文件夹到你的项目根目录即可。

### 使用方法

在 Claude Code 中，使用 Skill 工具调用学习助手，或者直接说明你需要学习帮助。

## 📚 使用示例

### 示例 1：解释代码

**你的问题：**
```
帮我解释这段 Python 代码：
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n-1)
```

**学习助手会：**
- 🎯 说明整体功能（计算阶乘）
- 📝 逐行解释代码逻辑
- 🔑 讲解递归的概念
- 💡 用生活例子类比（比如俄罗斯套娃）
- ⚠️ 提示常见错误和注意事项

### 示例 2：学习概念

**你的问题：**
```
什么是循环？为什么需要循环？
```

**学习助手会：**
- 📚 用简单语言定义循环
- 🎯 解释为什么需要循环（避免重复代码）
- 💻 提供实际代码示例
- 💡 用生活场景类比（比如洗碗、刷题）
- ✅ 展示正确用法和常见错误
