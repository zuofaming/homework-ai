# 🎓 Claude Code 学习助手技能包

一个专为编程初学者设计的 Claude Code Skill，帮助你用通俗易懂的方式学习编程、理解代码、掌握概念。

## ✨ 特点

- 📖 **通俗易懂**：用日常生活的比喻解释复杂的编程概念
- 🎯 **逐步引导**：从简单到复杂，循序渐进的教学方式
- 💡 **实用示例**：提供可直接运行的代码示例
- 🤝 **友好互动**：耐心、鼓励性的学习氛围
- 🔍 **深入浅出**：既讲"是什么"，也讲"为什么"

## 🚀 快速开始

### 安装方法

#### 方式一：直接下载（推荐）

1. 下载本仓库到本地：
```bash
git clone https://github.com/YOUR_USERNAME/claude-code-learning-skill.git
```

2. 复制 `.claude` 文件夹到你的项目根目录：
```bash
cp -r claude-code-learning-skill/.claude /path/to/your/project/
```

#### 方式二：手动创建

1. 在你的项目根目录创建以下结构：
```
your-project/
└── .claude/
    └── skills/
        └── learning-assistant/
            ├── skill.json
            └── skill.md
```

2. 复制本仓库中的 `skill.json` 和 `skill.md` 文件到对应位置

### 使用方法

在 Claude Code 中，使用 Skill 工具调用学习助手：

```
使用 learning-assistant 技能
```

或者直接在对话中说明你需要学习帮助。

## 📚 使用示例

### 示例 1：解释代码

**你的问题：**
```
帮我解释这段 Python 代码：
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n-1)
```

**学习助手会：**
- 🎯 说明整体功能（计算阶乘）
- 📝 逐行解释代码逻辑
- 🔑 讲解递归的概念
- 💡 用生活例子类比（比如俄罗斯套娃）
- ⚠️ 提示常见错误和注意事项

### 示例 2：学习概念

**你的问题：**
```
什么是循环？为什么需要循环？
```

**学习助手会：**
- 📚 用简单语言定义循环
- 🎯 解释为什么需要循环（避免重复代码）
- 💻 提供实际代码示例
- 💡 用生活场景类比（比如洗碗、刷题）
- ✅ 展示正确用法和常见错误

### 示例 3：调试帮助

**你的问题：**
```
这段代码报错了，为什么？
list = [1, 2, 3]
print(list[3])
```

**学习助手会：**
- 🔍 分析错误信息（IndexError）
- 📍 指出问题所在（索引越界）
- 📖 解释列表索引从 0 开始
- ✅ 提供修正方案
- 💡 教你如何避免类似错误

## 🌟 最佳实践

### 如何提出好问题

**✅ 好的提问方式：**
```
"帮我解释这段代码中的 map 函数是如何工作的"
"为什么我的循环没有停止？[附上代码]"
"Python 中的列表和元组有什么区别？给我一些例子"
```

**❌ 不够清晰的提问：**
```
"这个不work"（没有提供代码）
"教我编程"（范围太广）
"帮我写代码"（没说明需求）
```

### 学习技巧

1. **从小例子开始**：先理解简单的示例，再逐步增加复杂度
2. **动手实践**：看懂代码后，自己尝试修改和运行
3. **主动提问**：遇到不懂的地方立即询问
4. **总结归纳**：学完一个概念后，用自己的话总结
